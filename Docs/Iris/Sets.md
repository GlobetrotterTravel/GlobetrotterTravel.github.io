# Sets

A `set` is a named, dynamically generated collection of records belonging to a specific definition that is selected whenever it is accessed according to a predefined criteria. The criteria consists of a component definition to search for records in, and one or more facet values. Sets can be passed to filters just like a raw record collection, however sets have the advantage of being prefiltered. A set can contain a filtered list of another set, with unlimited levels of dependence.
